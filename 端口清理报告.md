## ⚡ 快捷命令

### 检查灵机系统端口范围

```bash
for port in 5173 5174 5175 5176; do \
  if lsof -Pi :$port -sTCP:LISTEN -t >/dev/null 2>&1; then \
    echo "❌ $port 被占用"; \
  else \
    echo "✅ $port 可用"; \
  fi; \
done
```

### 查看所有监听端口

```bash
lsof -iTCP -sTCP:LISTEN -P -n | grep -E ':(3000|5173|5174|5175)'
```
