## 模块依赖关系

```
M1 我的项目 ──────┬──→ M4 Tab A 授权片单 ──→ M8 手动回收弹窗
                  │                        ──→ M9 重新授权弹窗
                  ├──→ M5 Tab B 项目概览
                  ├──→ M6 Tab C 项目配置
                  └──→ M7 Tab D 新热预告

M2 中心审查池 ────┬──→ M3 审查单详情页
                  └──→ M10 添加确权IP弹窗
```

**执行顺序建议**（先做被依赖的）：
1. **第一批**：M1（我的项目）、M2（中心审查池）——两个一级页面，互相独立可并行
2. **第二批**：M3（审查详情）、M4（Tab A）、M5（Tab B）、M6（Tab C）、M7（Tab D）——依赖一级页面
3. **第三批**：M8（回收弹窗）、M9（重新授权弹窗）、M10（确权IP弹窗）——依赖上级页面
