| 2 | **审查池不需要场景 Tab**：二创权应用下只有二创授权一种场景 | M2 的 B1 场景 Tab 已删除 |
| 3 | **手动回收功能删除**：整体不做 | task_type 从 6 种缩减为 4 种 |

| 9 | **合同号格式 T180-PCG-\***：权益合同号格式为 T180-PCG-YYYYMMDD#####；一个合同号可对应多个权益 ID | mock 数据已更新 |

## 二、M1 我的项目 — 变更要点

- 项目卡片展示 **9 个字段**（项目名/权利类型/审核方式/状态/合作方/品类/时间/已授权数/已回收数）
- **3 维筛选**：权利类型(7选项) + 审核方式(3选项) + 状态(5选项)
- **创建项目 3 步弹窗**：选权利类型 → 选审核方式 → 填配置项
- 空态区分：初始无项目（"暂无项目，点击创建项目开始"）vs 筛选无结果（"暂无匹配项目"）
- deleted 状态项目不显示

## 三、M2 中心审查池 — 变更要点（改动最多）

| # | 列名 | 注意点 |
|---|------|--------|
| 1 | 审核ID | — |
| 2 | 作品名 | 独立列，不含别名 |
| 3 | **别名** | 独立列（一期是作品名下方副行） |
| 4 | 品类 | — |
| 5 | 版权ID | — |
| 6 | **权益合同号** | 展示引入+输出全部权益合同号（从 rights_infos 收集），**逐行展示**；格式 T180-PCG-*；一个合同号可对应多个权益ID |
| 7 | **热度** | 新增列，S/A/B/C 带颜色标签 |
| 8 | 任务时间 | — |
| 9 | 任务类型 | **只有 4 种**（授权审查/风险-分销冲突/风险-新增瑕疵/权益瑕疵重审），没有手动回收/重新授权 |
| 10 | 初审结论 | — |
| 11 | 审核进度 | 不含 obsoleted |
| 12 | 授权判定 | — |
| 13 | **信网权状态** | 新增列 |
| 14 | **正片在线** | 新增列 |
| 15 | 瑕疵类型 | — |
| 16 | 操作 | — |

> **原则：筛选器有的字段，表格中必须有对应列**

### 筛选器（10 个）
品类 / 任务类型 / 审核进度 / 复审结论 / **所属项目** / **热度口径** / **权益合同号** / **信网权状态** / **正片在线** / 关键字

### 批量操作（重点验收）
- 操作栏**常显**（不是勾选后才出现），**仅展示「批量终审」按钮**（初审和复审已删除）
- 批量终审弹窗顶部有**红色风险提示**
- 批量终审弹窗字段：**仅「授权判断」一个字段**（可授权/不授权），无备注，对齐单条详情页
- **终审风险提示**：授权起止时间默认采用系统填充时间，不支持单独修改
- **终审瑕疵校验**：复审结论为「权利瑕疵」的作品**不支持批量终审**（瑕疵处置结论无法统一填写），前置校验拦截并列出作品名
- **终审无「选择授权项目」**：二创场景从平台自动关联项目
- ~~批量初审~~（已删除：初审业务复杂度高，需逐条在详情页审核）
- ~~批量复审~~（已删除：复审业务复杂度高，需逐条在详情页审核）

## 四、M3 审查详情页 — 变更要点（对齐一期前端代码重写）

### 布局
- **左右分栏**：左侧流程节点进度条（flex:7），右侧当前阶段审核表单（flex:10）
- 已完成状态改为**纵向全节点回显**
- 审核历史按钮在**面包屑右侧**（不在作品信息卡内）

### 进度条
- **3 个节点**（初审 → 复审 → 运营授权评估），**没有「权益确认」节点**（confirmed_rights 是系统状态）
- 风险审查类型只有 2 步

### 三个审核表单（重点验收字段和顺序）

**初审**：结论 → 瑕疵类型(条件) → 可授权平台 → **排除平台**(新增) → 备注

**复审**（字段顺序严格对齐前端）：结论 → **授权依据(radio单选，非下拉)** → 可授权平台 → 排除平台 → 瑕疵类型(条件) → 瑕疵处置(条件) → 备注
- 授权依据：单权益时自动默认勾选

**终审**：授权判断 → **瑕疵处置结论**(新增，条件：复审为瑕疵) → 可授权平台(条件) → 授权时间(条件)

### 待审核信息（一期已有但 demo 之前缺失）
- 权益卡片列表：权益ID / 合同号 / 权益类型 / 风险标记
- 点击卡片展开**详情面板**（5大模块 radio 只读展示）：
  - 一、基本信息
  - 二、授权基础权利
  - 三、二创权利
  - 四、二创创作成果
  - 五、二创传播限制
