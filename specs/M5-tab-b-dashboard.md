# M5 执行规格 — 项目详情 Tab B：项目概览看板

> **模块编号**：M5
> **Rule 章节**：§三 3.4「Tab B 项目概览」
> **依赖**：M1
> **版本**：v1.0
> **显示条件**：仅多轮滚动审核项目（audit_mode=rolling）显示

---

## A. 页面定位

运营管理者在看板中一眼判断滚动授权项目的库存是否达到合同安全线。

---

## B. 字段清单

### B1. 看板模块定义（严格按 Rule §三 3.4 Tab B）

| 序号 | 模块名 | 内容 | Rule 依据 |
|------|--------|------|-----------|
| 1 | 核心指标 | 日均授权数量（大数字）+ 分品类日均（电视剧/电影/综艺/动漫/纪录片/少儿/微短剧） | §三 3.4 Tab B |
| 2 | 日均趋势（月度） | 按月的日均趋势线（实际 vs 预估-新热 vs 预计） | §三 3.4 |
| 3 | 日均趋势（季度/年度） | 按季度/年度的日均汇总 | §三 3.4 |
| 4 | 版权库存总览 | 双层堆叠柱状图（已授权+未授权储备），X轴品类，Y轴数量，3500安全线 | §三 3.4 |
| 5 | 已授权细分（饼图1） | 已授权 / 已授权-新热 / 已授权-需补确认函 | §三 3.4 |
| 6 | 未授权储备细分（饼图2） | 已复核 / 需补确认函 / 可用 / 权利瑕疵 | §三 3.4 |
| 7 | 分品类分布 | 按品类切换查看授权版权分布 | §三 3.4 |

### B2. 柱状图设计规范

| 属性 | 值 | Rule 依据 |
|------|---|-----------|
| 层数 | 2层（已授权/未授权储备），**不做更多细分** | §三 3.4 看板设计原则 |
| 已授权颜色 | 蓝色 | §三 3.4 |
| 未授权储备颜色 | 灰色 | §三 3.4 |
| 安全参考线 | 3500 部，红色虚线 | §三 3.4 |
| X轴 | 品类（7个） | §三 3.4 |

### B3. 饼图设计规范

**饼图1 — 已授权构成**（3种）：

| 类别 | 颜色建议 |
|------|---------|
| 已授权 | 蓝色 |
| 已授权-新热 | 浅蓝 |
| 已授权-需补确认函 | 橙色 |

**饼图2 — 未授权储备构成**（4种）：

| 类别 | 颜色建议 |
|------|---------|
| 已复核 | 绿色 |
| 需补确认函 | 橙色 |
| 可用 | 紫色 |
| 权利瑕疵 | 红色 |

### B4. 日均趋势设计

| 趋势线 | 含义 | Rule 依据 |
|--------|------|-----------|
| 实际日均 | 统计周期内每日有效授权版权数平均值 | §三 3.4 日均口径 |
| 预估-新热 | 加上即将满31天转正的新热后预估值 | §三 3.4 |
| 安全线 | 3500 部/日均 | §三 3.4 |

## C. 交互清单

| 序号 | 触发操作 | 预期结果 | Rule 依据 |
|------|---------|---------|-----------|
| 1 | 进入 Tab B | 渲染完整看板（7个模块） | §三 3.4 Tab B |
| 2 | 分品类分布切换品类 | 该品类的详细数据展示 | §三 3.4 |

> 看板以展示为主，交互较少。

---

## D. 边界条件

| 序号 | 条件描述 | 预期表现 |
|------|---------|---------|
| 1 | 非 rolling 项目 | Tab B 不出现在 Tab 栏 |
| 2 | 日均低于安全线 | 核心指标区域以红色/警示色标注 |
| 3 | 日均达到或超过安全线 | 核心指标区域以绿色标注 |

---

## E. 验收检查表

### 字段验证
- [ ] 核心指标：大数字 + 7 个品类日均
- [ ] 日均趋势：月度趋势图含 实际/预估-新热/安全线
- [ ] 柱状图：双层（蓝+灰），7品类，3500红色虚线安全线
- [ ] 饼图1：3种已授权构成
- [ ] 饼图2：4种未授权储备构成
- [ ] 分品类分布模块存在

### 交互验证
- [ ] 进入 Tab B → 看板完整渲染
- [ ] 分品类可切换

### 边界验证
- [ ] 非 rolling 项目 → 无 Tab B
- [ ] 日均 < 3500 → 红色/警示
- [ ] 日均 ≥ 3500 → 绿色

### 反向检查
- [ ] 柱状图只有 2 层，没有更多细分
- [ ] 安全参考线在正确的 Y 轴位置
- [ ] 无规格未定义的自创图表或指标
