# 灵机系统 - 使用说明

## ⚠️ 重要提示

**本项目不能直接双击HTML文件打开使用！**

由于项目使用了JavaScript `fetch()` API加载数据文件，浏览器的安全策略会阻止直接打开HTML文件时加载本地JSON文件。

---

## ✅ 正确的使用方法

### 方法1：Python本地服务器（推荐，最简单）

1. 打开「终端」（Terminal）或「命令提示符」
2. 进入项目文件夹：
   ```bash
   cd /path/to/IP权益变现应用_20260107
   ```
3. 启动服务器：
   ```bash
   python3 -m http.server 8080
   ```
   或者（如果上面命令不行）：
   ```bash
   python -m http.server 8080
   ```
4. 打开浏览器访问：**http://localhost:8080**

**停止服务器**：在终端按 `Ctrl + C`

---

### 方法2：使用VS Code Live Server扩展

1. 使用VS Code打开项目文件夹
2. 安装「Live Server」扩展
3. 右键点击`index.html`，选择「Open with Live Server」
4. 自动在浏览器中打开

---

### 方法3：使用Node.js的http-server

1. 先安装http-server（只需第一次）：
   ```bash
   npm install -g http-server
   ```
2. 进入项目文件夹：
   ```bash
   cd /path/to/IP权益变现应用_20260107
   ```
3. 启动服务器：
   ```bash
   http-server -p 8080
   ```
4. 打开浏览器访问：**http://localhost:8080**

## ❓ 常见问题

### Q: 为什么不能直接双击HTML打开？
A: 项目使用`fetch()`加载JSON数据文件，浏览器的CORS安全策略会阻止这种操作。必须通过HTTP服务器访问。

### Q: 启动服务器后看不到素材查询功能？
A: 
1. 确认浏览器地址是 `http://localhost:8080`（不是 `file://`）
2. 点击左侧菜单「商业化素材库」→「素材查询与申用」
3. 按F12打开浏览器控制台查看是否有错误信息

### Q: 端口8080被占用怎么办？
A: 可以使用其他端口，例如：
```bash
python3 -m http.server 3000
```
然后访问 `http://localhost:3000`
