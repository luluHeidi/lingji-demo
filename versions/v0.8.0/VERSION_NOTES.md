### 2️⃣ **智能搜索系统**

#### 三种搜索模式

**名称模糊搜索（AI关闭）**
- 输入示例："海报"
- 搜索方式：模糊匹配包含关键词的素材
- 筛选器：显示
- 结果：多个相关素材

**ID精准搜索（AI关闭）**
- 输入示例："SC20260119MX01"
- 搜索方式：精准匹配该ID
- 格式识别：`/^SC\d{14}$/`正则自动识别
- 结果：单个精准素材

**AI智能搜索（AI开启）**
- 输入示例："请给我找一下长相思可以用于制作人物小卡的素材"
- 搜索流程：
  1. AI思考加载动画（800ms）
  2. 打字机效果显示分析文案（30ms/字）
  3. 智能推荐素材
  4. AI全选按钮自动显示
- 筛选器：自动隐藏
- 结果：智能推荐+分析报告

#### AI交互优化
- **AI开关切换**：界面无变化（避免突兀跳变）
- **分析文案结构化**：需求分析 + 推荐策略
- **打字机效果**：30ms/字，闪烁光标动画
- **加载动画**：24×24px旋转图标

## 📋 使用指南

### 如何访问模块
1. 打开主页面 `index.html`
2. 点击左侧菜单"商业化素材库"
3. 点击"素材查询与申用"
4. 进入模块主页

### 功能使用流程

**流程1：快速查找素材**
```
1. 选择品类（电视剧/动漫/综艺）
2. 使用筛选器（状态/等级）
3. 输入关键词搜索
4. 点击IP卡片进入素材详情
```

**流程2：AI智能搜索**
```
1. 打开AI开关
2. 输入需求描述（如"请给我找一下长相思可以用于制作人物小卡的素材"）
3. 点击搜索按钮
4. 查看AI分析和推荐结果
```

**流程3：批量申用素材**
```
1. 进入IP素材详情页
2. 使用筛选器缩小范围
3. 勾选需要的素材（或使用全选按钮）
4. 点击"批量申用"按钮
5. 填写申用表单
6. 提交申用
```
